# HJB CodeForge — Website

Official website for [HJB CodeForge](https://hjbcodeforge.com).

**Powerful software, precisely built.**

## Stack

- Pure HTML / CSS / JavaScript — no build step required
- Hosted on Vercel
- Custom domain: hjbcodeforge.com

## Pages

- Home
- Services
- About
- Portfolio
- Blog
- Contact

## Deployment

This site deploys automatically via Vercel on every push to `main`.

To run locally, simply open `index.html` in your browser — no server needed.

## Updates

- To add a blog post: add a new `.post-card` block in the blog section of `index.html`
- To add a portfolio project: add a new `.project-card` block in the portfolio section
- To update contact details: search for `hello@hjbcodeforge.com` in `index.html`
