# HJBCodeForge Website

This is the official business website for HJBCodeForge, a software development company specializing in custom web and mobile solutions.

## Project Structure

The project is organized with a clean separation of concerns:

- `index.html`: The main HTML file containing the structure and content of the website.
- `styles.css`: The primary stylesheet for the website's layout and design.
- `animations.css`: A dedicated stylesheet for animations and transitions.
- `script.js`: The JavaScript file for all dynamic functionality and interactivity.
- `images/`: A directory containing all image assets, including the company logo and icon.

## How to Run

To view the website, simply open the `index.html` file in your web browser.

## 🛠️ Setup Instructions

1. **Clone or download** the project files
2. **Ensure all files** are in the same directory structure as shown above
3. **Open index.html** in a web browser or serve through a local server

### Local Development Server (Recommended)

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

## 🔧 Customization

### Updating Company Information
- Edit text content in `index.html`
- Update contact details in the contact section
- Modify the company description in the about section

### Changing Colors
- Update CSS custom properties in `styles.css` (`:root` section)
- Modify the color scheme to match your branding

### Adding Content
- Add new portfolio items in the portfolio section
- Include additional services as needed
- Update testimonials or client information

### Images
- Replace logo files in the `images/` directory
- Ensure new images maintain similar dimensions
- Update image references in HTML if filenames change
